# 🔒 MYCHA VAULT-STORAGE // Digital Private Vault

A modern, high-performance web-based private storage management system with a interactive unlock screen, real-time file filtering, storage monitoring, and seamless GitHub integration.

---

## 🚀 Live Application

- **URL:** [https://gbollzgee.github.io/vault-storage/](https://gbollzgee.github.io/vault-storage/)

---

## ✨ Features & Highlights

- **Interactive Entrance Security UI:** Custom interactive pull-cord mechanic to unlock and sign into the private vault interface.
- **File Management & Organization:**
  - Dynamic file categorization across media, documents, and assets.
  - Folder creation and custom folder filtering.
  - Multi-criteria sorting options (Newest, Oldest, Alphabetical A–Z, Size).
- **Storage Monitoring:** Live storage meter tracking GitHub usage with built-in alert thresholds (e.g., approaching 1 GB storage limit).
- **Search & Filtering:** Real-time search bar and filter controls for quick asset navigation.
- **Modal Dialog Workflows:**
  - File detail preview and meta management.
  - Folder creation modal.
  - Delete confirmation dialogs to prevent accidental file removal.
- **Responsive & Accessible Design:** Built using accessible ARIA roles, smooth CSS transitions, and mobile-friendly layouts.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **UI/UX:** CSS Grid/Flexbox, Custom Keyframe Animations, Accessible ARIA Components
- **Storage Backend:** GitHub API Integration / Web Storage API

---

## 💻 How It Works

1. **Vault Entrance:** Pull the interactive unlock mechanism on the entrance screen to open the sign-in modal.
2. **Access Dashboard:** View all stored files, monitor GitHub storage allocation, or filter items by custom folders.
3. **File Operations:** Search, sort, or open individual file modal previews to inspect metadata or manage assets.
